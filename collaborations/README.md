## 🔄 **협업 동기화 방법**

### 자동 동기화 (권장)
GitHub Actions가 매주 자동으로 협업 파트너들의 코드를 동기화합니다.

### 수동 동기화
```bash
# KLP-SignGlove 업데이트
cd collaborations/KLP-SignGlove
git pull origin main

# SignGlove_HW 업데이트
cd ../SignGlove_HW
git pull origin main

# SignGlove-DataAnalysis 업데이트
cd ../SignGlove-DataAnalysis
git pull origin main
```

2. **정기적인 소통**: 주간/월간 진행상황 공유
3. **코드 품질 관리**: 코드 리뷰 및 테스트 필수
4. **문서화**: 협업 과정과 결과 기록
